title: QEMU网络模式-nat
---

主要功能是：建立bridge,设置brdige的内网IP,并且将客户机的网络接口与其绑定，然后打开系统中网络IP包转发的功能，设置iptables的NAT规则，最后启动一个DHCP服务。

**建立bridge**

```shell
brctl addbr br0
brctl stp br0 on
# brctl setfd br0 0
ip link set dev br0 up
```

**建立客户机网络接口并绑定网桥**

```shell
tunctl -u $(whoami) -t tap0
brctl addif br0 tap0
```

**打开系统转发功能**

```shell
echo 1 > /proc/sys/net/ipv4/ip_forward
```

**设置iptables规则**

```shell
iptables -t nat -A POSTROUTING -s 192.168.1.0/24 ! -d 192.168.1.0/24 -j MASQUERADE
```

**启动DHCP服务**

```shell
systemctl restart isc-dhcp-server
```

假定主机IP：10.0.2.15，客户机IP：192.168.1.2，添加iptables规则进行端口映射，让外网主机也能访问客户机。

```shell
iptables -t nat -A PREROUTING -p tcp -d 10.0.2.15 --dport 80 -j DNAT --to 192.168.1.2:80
```
